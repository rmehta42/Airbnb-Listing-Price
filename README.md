# NYC Airbnb Analysis

This project analyzes Airbnb listing data in New York City to identify trends in pricing, availability, neighborhood activity, and listing characteristics. The goal of the analysis was to explore how factors such as borough, room type, minimum nights, and review activity influence listing prices and market behavior across NYC.

## Project Objectives
- Clean and preprocess a large real-world dataset
- Perform exploratory data analysis on Airbnb listings
- Identify pricing and availability trends across boroughs
- Visualize relationships between listing characteristics and price
- Examine disparities in listing concentration and market activity

## Tools and Technologies
- Python
- pandas
- NumPy
- matplotlib
- seaborn
- scikit-learn
- Jupyter Notebook
- SQL

## Dataset
The dataset contains Airbnb listing information for New York City, including:
- Listing price
- Borough and neighborhood
- Room type
- Availability
- Number of reviews
- Minimum nights required
- Host information

## Methodology
The project involved:
1. Data cleaning and preprocessing
2. Handling missing and inconsistent values
3. Exploratory data analysis and visualization
4. Statistical analysis of pricing and availability
5. Identifying trends and relationships within the dataset

## Key Findings
- Manhattan listings had the highest median prices across all boroughs.
- Entire home/apartment listings were significantly more expensive than private rooms.
- Certain neighborhoods showed high listing density combined with low yearly availability.
- Review activity and room type demonstrated measurable relationships with pricing patterns.
